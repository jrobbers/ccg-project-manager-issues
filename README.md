# 🐛 CCG Project Manager (waypoint) - Issue Tracker

This is the **public issue tracker** for the CCG Project Manager. 

## 📝 How to Report Issues

We appreciate bug reports and feature requests from our community! To keep things organized, please use one of our issue templates:

### 🐛 [Report a Bug](https://github.com/jrobbers/ccg-project-manager-issues/issues/new?template=bug_report.yml)
Found something that doesn't work? Let us know! Please include:
- Description of the bug
- Steps to reproduce
- Expected vs actual behavior
- Environment details (OS, browser, version)
- Screenshots (if applicable)

### ✨ [Request a Feature](https://github.com/jrobbers/ccg-project-manager-issues/issues/new?template=feature_request.yml)
Have a great idea? Share it with us! Please include:
- Clear description of the feature
- Use case and why it's useful
- Proposed solution
- Priority level

## 📖 Guidelines

- **Search first** - Check if your issue already exists
- **Be descriptive** - The more detail, the faster we can help
- **One issue per report** - Keep issues focused and specific
- **Be respectful** - We're all here to help each other
